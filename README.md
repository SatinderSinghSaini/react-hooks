# react-hooks
Created this project using Next.js(Server side rendering of react)
Steps to initialize the Repo:
  1.  create folder
  2.  open git bash in this folder
  3.  npm init -y
  4.  add below scripts in the package.json
        "dev": "next",
        "build": "next build",
        "start": "next start"
  5.  npm i react react-dom next
